Tutorials
=========

All tutorials are written as `Jupyter notebooks <http://jupyter.org>`__.
To run a notebook locally, download it by clicking on "Download" at the top of
the page (you might need to right click and select "Save link" or something
similar).

You can also try a demo notebook online (no install required) at
`try.gmtpython.xyz <http://try.gmtpython.xyz>`__
